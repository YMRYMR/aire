## Structure of Help Items

Each help item may contain the following fields:

- `tab` ‚Äì Section category (e.g., "Setup Wizard", "Getting Started"). **Do not translate**.
- `type` ‚Äì Content type: "text", "table", or "code". **Do not translate**.
- `title` ‚Äì Short heading (translate).
- `content` ‚Äì Main explanatory text (translate; may contain bullet points ‚Ä¢ and placeholders like `{0}`).
- `links` ‚Äì Array of hyperlinks; translate the `label` field, keep `action` unchanged.
- `imagePath` ‚Äì Path to image file (do not translate).
- `imageCaption` ‚Äì Caption for an image (translate).
- `intro` ‚Äì Introductory text before a table (translate).
- `cols` ‚Äì Table column headers (translate).
- `rows` ‚Äì Table rows, each cell may contain text (translate).

## Translation Guidelines

1. **Preserve placeholders**: Any `{0}`, `{1}`, etc. must remain exactly as they appear.
2. **Bullet points**: The bullet character `‚Ä¢` should be kept; you may replace with a localized bullet if appropriate.
3. **Structural fields**: Do not translate `tab`, `type`, `action`, `imagePath`.
4. **URLs and technical terms**: Keep brand names (OpenAI, Anthropic, Ollama, etc.) untranslated unless there is a widely accepted localized name.
5. **Consistency**: Use the glossary to ensure consistent translation of recurring terms like "provider", "model", "tool", etc.
